Developers' info:
To build the FireFox extension:
1. Install Java JDK, make sure JAVA_HOME and PATH are able to find Java:
   On Windows: Start -> Run -> cmd -> Ok -> javac -> ENTER
               Start -> Run -> cmd -> echo %JAVA_HOME%
   Mac OS X/Linux: type javac and echo $JAVA_HOME
2. Install Apache Ant, make sure ANT_HOME and PATH are able to find Ant:
   On Windows: Start -> Run -> cmd -> Ok -> ant -> ENTER
               Start -> Run -> cmd -> echo %ANT_HOME%
   Mac OS X/Linux: type ant and echo $ANT_HOME
3. type ant
